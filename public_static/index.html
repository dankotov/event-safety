Popularity

In 2019, React is one of (if not the most) popular JavaScript frontend framework. According to StackOverflow's 2018 surveys, React ranks #3 in most popular. Per the stateofjs.com survey for 2018, React ranks highly on amount of users that have used it and would use it again: 64.8% used it and would use it again and only 6.7% of users would not use it again.

Usage

React excels because it’s pure JavaScript. If you have a team of JavaScript developers that know the ins and outs of the language, then it’s a great fit. JavaScript developers will be able to embrace JS, build components with pure JavaScript and understand styling with CSS-in-JS.


Advantages

  • Clean programming, strong community, fast performance

Disadvantages

  • Large library, learning curve, view-orientedness

Popularity

Express.js is the most popular node framework as evidenced by its attainment of 43,000 GitHub stars. This is due to the framework’s unopinionated, fast, and minimalist design philosophy. Express has become so popular to the extent that it is synonymous with Node.js. Express is also used by Uber, IBM, and PayPal.

Usage

Express functions as a middleware framework that sets up routes to receive and send requests between the front-end and the database of a web application. A chat-based application is the most common use case for Node and Express because chat applications are high traffic, data-intensive, and lightweight.


Advantages

  • The standard framework for Node.js web middleware, fully customisable, open source

Disadvantages

  • Single threaded framework, relies on the asynchronous programming model, weak library support system
