React & Express Installation Guide (Ubuntu)

In order to use React, we need to install Node.js as our server environment.

To install Node.js open your terminal and type: 

$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -


This command adds the NodeSource signing key to your system and installs all necessary packages.

Once the NodeSource repository is enabled, install Node.js and npm by typing: 

$ sudo apt install nodejs


Install create-react-app on your computer with NPM as global using the below command. 

$ npm install -g create-react-app


After the successful installation of create-react-app, we can create our first React application using it.

$ create-react-app hello-react


This command creates a new project folder named hello-react.

You can install express.js using npm. The following command will install latest version of express.js globally on your machine.

$ npm install -g express


The instructions for installing React on other operating systems can be found in the link below:

https://react-cn.github.io/react/downloads.html
